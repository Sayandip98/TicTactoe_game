# 🎮 Tic-Tac-Toe

A classic, interactive Tic-Tac-Toe web game built with Vanilla JavaScript, HTML5, and CSS3. Designed with a clean and responsive user interface, it provides an engaging local two-player experience.

## ✨ Features

- **Interactive Gameplay**: Play the traditional game of Tic-Tac-Toe with immediate visual feedback.
- **Winner Detection & Draw Handling**: Automatically detects the winning conditions or declares a draw if all boxes are checked.
- **Responsive Design**: Adapts gracefully to both desktop and mobile screens using relative `vmin` sizing.
- **Modern UI**: Clean minimalist aesthetics with pleasant color palettes, curved borders, and subtle box-shadows.
- **Reset & Restart**: Functional "Reset Game" and "New Game" buttons to quickly clear the board and start a fresh round.

## 🛠️ Tech Stack

- **HTML5**: Semantic tags and simple container structure.
- **CSS3**: Flexbox layout, responsive sizing variables (`vmin`, `vh`), and UI styling.
- **Vanilla JavaScript ES6**: DOM manipulation, event listeners, and game logic tracking (turns, win patterns, draw states).

## 🚀 Getting Started

To run this project locally, no complex setup or server is required.

1. **Clone the repository** (or download the source code):
   ```bash
   git clone https://github.com/Sayandip98/TicTactoe_game.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd TicTactoe_game
   ```
3. **Run the game**:
   Simply open the `index.html` file in any modern web browser.

## 📂 Project Structure

- `index.html` — The main structure of the game application.
- `style.css` — The styling sheet providing visually appealing aesthetics and responsive design.
- `app.js` — The core logic, click handlers, and win-condition validations.
